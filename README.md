## Project Title

Side Quest Week 6: Cat Fighter Platformer.

---

## Description

> This project lets users interact with a 2-dimensional side-scroller game using keyboard inputs: moving around, 'rescuing' plants, fighting enemies, and avoiding obstacles.  
> It explores the implementation of both visual and audio assets with p5 play.

---

## Interaction Instructions

> Press W/A/D or the left, right, and up arrow keys to move player character around.
> Press space bar to attack.
> Progress to the right of screen while collecting ('rescuing') the plants - a total of 15, tracked with a counter on the upper left corner - to win.
> If taking damage, both player character and enemies will display 'hurt' sprite state while receiving knockback and flashing red (invincibility frames). Player health is also displayed through three hearts on the upper right corner of the screen.
> When dead, both player character and enemies will display 'death' sprite animation, after which a 'You Died' screen is displayed along with instructions to press 'R' to return to beginning.
> A timer shows on screen for how long users played once a round ends.
> Plants disappear when is 'rescued' (collided with player character).
> Background music plays throughout while sound effects are played for jumping, hitting the enemies, collecting the plants, as well as receiving damage.
